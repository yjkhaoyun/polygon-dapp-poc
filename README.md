### 使用时需要将polygon测试网添加进小狐狸
参考 https://wiki.polygon.technology/docs/develop/network-details/network/

```
网络名称：      polygon-mumbai
测试网RPC URL： https://rpc-mumbai.matic.today
链ID：          80001
币种符号：      MATIC
区块浏览器URL：	https://mumbai.polygonscan.com/
```
